HTML Block and Inline Elements
Block-level Elements：
A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).

Inline Elements：
An inline element does not start on a new line and only takes up as much width as necessary.

关于div和span标签，div经常是作为其它HTML元素的容器，span经常作为其它文本的容器。两者都没有需要的属性，但是 style class id是通用的。通常和CSS配合，来装饰元素的呈现方式。
class属性
The class attribute specifies one or more class names for an HTML element.
The class name can be used by CSS and JavaScript to perform certain tasks for elements with the specified class name.
In CSS, to select elements with a specific class, write a period (.) character.
JavaScript can access elements with a specified class name by using the getElementsByClassName() method.
HTML元素可以有多个class 中间空格分割。
id属性
类似class，我们用id来唯一标识一个HTML元素，使用#去选择一个拥有特定id的元素。
id值至少包含一个字符，并且中间不允许包含空白。
JavaScript can access an element with a specified id by using the getElementById() method.
