CSS学习笔记

Apr 2, 2018

CSS stands for Cascading Style Sheets（层叠样式表）
CSS Syntax and Selectors
1.element selector:基于元素
2.id selector:用HTML中的id属性(唯一的)选择，#idName
3.class selector:类 .className 或者 elementName.className
4.grouping selectors: 如 p,h1,h2{ Declaration }
5.注释：/ comments /
How To
1.External Style Sheet：使用<link>放于<head>中；
2.Internal Style Sheet：置于<style>中，放在<head>中；
3.Inline Styles：利用HTML元素的style属性
Backgrounds
1.background-color
2.background-image:url(img_url)
3.background-repeat:no-repeat repeat-x repeat-y (defalut为铺满)
4.background-attachment:fixed scroll(default)
5.background-position:top right left center
Boders
Allow us to specify the style, width, and color of an element’s border.
Shorthand Property:border-width border-style (required) border-color
border-radius:5px(使边缘变得圆滑)
Margins
Margin properties are used to create space around elements, outside of any defined borders.
Shorthand Property:margin-top margin-right margin-bottom margin-left
Padding
Padding properties are used to generate space around an element's content, inside of any defined borders.对其中的内容加上空白边距。
Shorthand Property:类似Margin
利用box-sizing: border-box解决Padding和Element Width的累加。
Box Model
The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
Important:When you set the width and height properties of an element with CSS, you just set the width and height of the content area. To calculate the full size of an element, you must also add padding, borders and margins.
Outline
An outline is a line that is drawn around elements, OUTSIDE the borders, to make the element “stand out”.
Shorthand property:outline-width outline-style (required) outline-color
Text
1.text-align:left right center justified
2.text-decoration property is used to set or remove decorations from text.
常用a{text-decoration:none;}去除链接的下划线。
3.text-transform property is used to specify uppercase and lowercase letters in a text.
uppercase lowercase capitalize
4.letter-spacing property is used to specify the space between the characters in a text.
5.line-height property is used to specify the space between lines
6.text-shadow property adds shadow to text.
7.text-indent:首行缩进
Fonts
1.font-family；
2.font-style:normal italic oblique
3.font-weight / font-size
Icons
最简单添加icon进HTML页面的方式是使用icon Library，例如Font Awesome
Links
Links can be styled with any CSS property (e.g. color, font-family, background, etc.).
a:hover MUST come after a:link and a:visited
a:active MUST come after a:hover
顺序 link visited hover active
Lists
1.list-style-type;
2.list-style-image:url(‘ ‘);
3.list-style-position: inside/outside
Shorthand property:list-style-type list-style-position list-style-image
Display
The display property specifies if/how an element is displayed.
display:none block inline
Block-level Elements：
div h1-h6 p form header footer section
A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
一个block-level的元素总是从一个新的行开始，并且尽可能地占据整个宽度。
An inline element does not start on a new line and only takes up as much width as necessary.
span a img

Position
1.static:is not positioned in any special way; it is always positioned according to the normal flow of the page.
2.relative:is positioned relative to its normal position.使用left right top bottom改变位置。
相对于原来正常位置进行改变，其他元素位置不变。
3.fixed:is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled.窗口中固定位置。
4.absolute: is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).相对于最近父亲位置，并不会占据Normal时的位置。
5.sticky:is positioned based on the user’s scroll position.position: -webkit-sticky; / Safari /
6.使用z-index解决overlapping问题。
Overflow
Controls what happens to content that is too big to fit into an area.
可以进行选择是裁剪内容或者加上scroll来解决将内容放入特定大小区域。
Properties:visible hidden scroll auto
Note:The overflow property only works for block elements with a specified height.
overflow-x overflow-y
CSS Layout - float and clear
float properties: right left none inherit
In its simplest use, the float property can be used to wrap text around images.
The clear property specifies what elements can float beside the cleared element and on which side.
Propertie:left right none both inherit
Align
margin:水平居中
padding & text-center 使得水平和竖直居中
使用position:absolute left right:0px 实现居中 / flow：right/left
Combinators
There are four different combinators in CSS:
1.descendant selector (space) 后代选择器
2.child selector (>) 直接的孩子
3.adjacent sibling selector (+) 可选择紧接在另一元素后的元素，且二者有相同父元素。
4.general sibling selector (~) 所有拥有相同父元素，且跟在该元素后面的元素。
Pseudo-classes
A pseudo-class is used to define a special state of an element.
伪类被用来定义一个元素的特殊状态。
For example, it can be used to:
1.Style an element when a user mouses over it
2.Style visited and unvisited links differently
3.Style an element when it gets focus
语法：selector:pseudo-class { property:value; }
Anchor Pseudo-classes：
a:link a:visited a:hover a:active
出于效率的考虑：a:hover置于a:link & a:visited之后，a:active置于a:hover之后。
Hover on div
div:hover { background-color: blue; }
The :first-child pseudo-class matches a specified element that is the first child of another element.
The :lang pseudo-class allows you to define special rules for different languages.
所有伪类
Pseudo-elements
A CSS pseudo-element is used to style specified parts of an element.
For example, it can be used to:
1.Style the first letter, or line, of an element
2.Insert content before, or after, the content of an element
Syntax:selector::pseudo-element { property:value; }
The ::first-line pseudo-element can only be applied to block-level elements.
The ::selection pseudo-element matches the portion of an element that is selected by a user.
Opacity / Transparency
filter:alpha(opacity=x). The x can take a value from 0 - 100.
All of its child elements inherit the same transparency.
background: rgba(76, 175, 80, 0.3)不会影响其中的孩子元素。
